# Super Store Sales Analysis & Forecast Dashboard

## Project Overview

This project focuses on analyzing Super Store sales data and building an interactive business intelligence dashboard using :contentReference[oaicite:0]{index=0}.

The dashboard provides insights into:
- Sales performance
- Profit analysis
- Regional trends
- Customer segments
- Product category performance
- 15-Day sales forecasting

The project demonstrates end-to-end data analysis and dashboard development skills required for Data Analyst and Business Analyst roles

## Key Business Insights

- Identified top-performing regions and states
- Analyzed profitable customer segments
- Compared sales across product categories
- Evaluated shipping mode performance
- Tracked monthly sales and profit trends
- Predicted future sales using forecasting models
